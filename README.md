
## K2.Tasks-Service是什么?
K2.Tasks后端处理程序。


## K2.Tasks-Service有哪些功能？

* 提供API服务，包含：身份认证、待办任务获取、流程审批、任务推荐API。
* 实现K2EventListener功能，该功能实现实时待办任务推送。
